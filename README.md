# BecSupport

An Android application for BEC College students to download question papers, syllabus, results and share their question papers.

### Skills Used:
1. Java
2. XML
3. Firebase (Google signup, Realtime Database, Cloud Storage, Cloud Message)
4. Mysql
5. php
6. Json
